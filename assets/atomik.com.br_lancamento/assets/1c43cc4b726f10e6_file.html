Política de Privacidade

Data da última atualização: 20/05/2026

1

Visão Geral

O Atomik é uma extensão para o Google Chrome que aprimora o editor Elementor no WordPress. Esta Política de Privacidade explica quais dados coletamos, como os utilizamos e quais são os seus direitos em relação a essas informações.

Ao instalar e utilizar o Atomik, você concorda com as práticas descritas nesta política. Caso não concorde, por favor desinstale a extensão.

2

3

Onde o Atomik atua

O Atomik injeta funcionalidades apenas nas seguintes páginas para entregar seus recursos:

- Editor Elementor (/wp-admin/post.php?action=elementor), para inserir componentes e aplicar configurações de design diretamente no editor

- Painel Administrativo do WordPress (/wp-admin/*), para habilitar recursos de automação do WordPress, como gerenciamento de plugins e configurações do site

- Figma (figma.com). para ler dados de design dos próprios arquivos Figma do usuário e importá-los para o Elementor

- Conta Elementor (my.elementor.com), para integração com a conta e licença do Elementor


A extensão não é ativada em nenhum outro site e não coleta dados de páginas fora desses contextos.

O Chrome pode exibir os indicadores de acesso a "Fontes", "Rede local" e "Apps no dispositivo" ao instalar/utilizar a extensão. Isso ocorre porque: (1) o Atomik lê informações de fontes do DOM das páginas para listar as tipografias disponíveis no site do usuário; e (2) a permissão de acesso a sites via http:// inclui servidores locais de desenvolvimento WordPress (localhost, 192.168.x.x), que muitos desenvolvedores utilizam para testar seus projetos. Nenhum dado de fontes ou de rede local é transmitido aos servidores do Atomik.

5

O que NÃO coletamos

- Histórico de navegação ou atividade fora do Elementor, painel WordPress e Figma

- Conteúdo dos sites que você visita

- Informações pessoais além do necessário para a validação da licença

- Análises de comportamento, rastreamento ou telemetria de uso

- Identificadores publicitários de qualquer tipo

6

Como os dados são utilizados

- Os dados de licença são usados exclusivamente para verificar o acesso ativo e impedir uso não autorizado

- As preferências do usuário são usadas para restaurar suas configurações entre sessões e dispositivos

- Tokens do Figma, credenciais WordPress e chaves de API de IA são utilizados exclusivamente para executar as ações que você inicia — nunca são lidos para nenhuma outra finalidade

- Não vendemos, compartilhamos nem transferimos seus dados a terceiros para fins publicitários ou de marketing

7

Base Legal para o Tratamento (LGPD)

O tratamento dos seus dados pessoais é realizado com base nas seguintes hipóteses previstas na Lei Geral de Proteção de Dados (Lei nº 13.709/2018):

- Execução de contrato: dados de licença e e-mail são necessários para entregar o serviço contratado

- Legítimo interesse: preferências e dados de uso são necessários para o funcionamento correto da extensão

- Consentimento: dados inseridos voluntariamente (tokens Figma, credenciais WP, chaves de IA) são tratados com base no consentimento expresso do usuário ao inseri-los

8

Serviços de terceiros

Supabase

Utilizamos o Supabase (supabase.com) para armazenar registros de licença e backups de preferências do usuário. Os dados são hospedados em servidores na América do Sul (São Paulo, Brasil). Política de privacidade do Supabase: supabase.com/privacy

Hotmart

As compras são processadas pela Hotmart (hotmart.com). Ao adquirir o Atomik, a Hotmart compartilha seu endereço de e-mail e código de transação conosco para fins de ativação da licença. Política de privacidade da Hotmart: hotmart.com/legal/privacy-policy

API do Figma

O recurso de importação Figma se comunica diretamente com a API do Figma utilizando o token de acesso fornecido pelo usuário. O Atomik não armazena nem transmite esse token para nossos servidores. Política de privacidade do Figma: figma.com/privacy

Google Fonts

A interface da extensão carrega fontes do Google Fonts, o que envolve uma requisição padrão aos servidores do Google. Política de privacidade do Google: policies.google.com/privacy

9

Armazenamento e Segurança dos Dados

Preferências e dados em cache são armazenados localmente usando a API chrome.storage do Chrome e não são acessíveis por outras extensões ou sites. O backup de preferências não sensíveis é armazenado no Supabase, protegido por políticas de segurança em nível de linha (Row Level Security) que garantem que cada usuário acesse apenas seus próprios dados.

Credenciais sensíveis (tokens Figma, senhas WordPress, chaves de API de IA) são armazenadas exclusivamente no seu navegador local e jamais são transmitidas aos servidores do Atomik.

Adotamos medidas técnicas e organizacionais adequadas para proteger seus dados contra acesso não autorizado, alteração, divulgação ou destruição.

10

Retenção dos Dados

Os registros de licença são mantidos pelo período de vigência da licença e por até 12 meses após o encerramento, para fins de suporte. O backup de preferências do usuário é mantido enquanto a licença estiver ativa.

Mediante solicitação, todos os dados armazenados em nossos servidores associados à sua conta podem ser excluídos — veja a Seção 12.

Os dados armazenados localmente no seu navegador podem ser removidos a qualquer momento desinstalando a extensão.

11

Privacidade de Menores

O Atomik não é destinado a menores de 13 anos e não coletamos intencionalmente dados pessoais de crianças. Caso acredite que uma criança forneceu informações pessoais através da extensão, entre em contato conosco e os dados serão excluídos prontamente.

12

Seus Direitos (LGPD)

Nos termos da LGPD, você tem direito a:

- Confirmar a existência de tratamento dos seus dados

- Acessar os dados pessoais que mantemos sobre você

- Corrigir dados incompletos, inexatos ou desatualizados

- Solicitar a anonimização, bloqueio ou eliminação de dados desnecessários ou excessivos

- Solicitar a portabilidade dos seus dados

- Revogar o consentimento a qualquer momento

- Opor-se ao tratamento realizado com base em legítimo interesse

Para exercer qualquer desses direitos, entre em contato pelo e-mail [email protected]. Responderemos em até 15 dias úteis.

13

Transferência Internacional de Dados

Alguns dos serviços de terceiros utilizados (Supabase, Figma, Google Fonts) podem processar dados em servidores localizados fora do Brasil. Quando isso ocorre, garantimos que tais transferências estejam em conformidade com a LGPD e que os serviços adotem nível adequado de proteção de dados.

14

Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Quando isso ocorrer, atualizaremos a data de "Última atualização" no topo desta página. O uso continuado da extensão após as alterações implica na aceitação da política atualizada. Em caso de alterações relevantes, notificaremos os usuários dentro da própria extensão.
