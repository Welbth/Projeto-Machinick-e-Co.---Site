O Atomik substitui o Elementor? Preciso instalar algo no WordPress?

Não, o Atomik é uma extensão para o Google Chrome que roda dentro do editor do Elementor. Ela adiciona componentes, design system, animações e importação do Figma sem você precisar instalar nada no WordPress.

Posso usar em quantos sites eu quiser?

Sim. Uma licença Atomik libera o uso em quantos sites você quiser — só precisa estar logado na extensão ao abrir o editor do Elementor no seu navegador.

O Atomik é uma assinatura? Qual é o tempo de acesso?

Sim, o Atomik é uma assinatura anual.
